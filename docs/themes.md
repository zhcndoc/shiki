## 特殊的主题

你可以将主题设置为 `none` 以绕过高亮显示。这可以作为处理用户指定了不可用主题时的回滚效果，例如：

```ts twoslash theme:none
import { codeToHtml } from 'shiki'

const html = codeToHtml('console.log("Hello World")', {
  lang: 'javascript',
  theme: 'none', // [!code hl]
})
```
