# @shikijs/magic-move

Smoothly animated code blocks with [shiki](https://github.com/shikijs/shiki). Useful for code-step animations in slide decks (e.g. [Slidev](https://sli.dev/guide/syntax#shiki-magic-move)) and tutorials.

[Documentation](https://shiki.style/packages/magic-move)
